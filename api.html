BFDI Character API

Access detailed information about all Battle for Dream Island characters in JSON format!

What is this?

This API provides structured data about BFDI characters, including their names, descriptions, badges, and image paths. Perfect for developers building BFDI-related projects or fans who want to explore character data programmatically.

Features

  • 60+ BFDI characters with detailed information
  • Character names, descriptions, and roles
  • Image file paths for each character
  • Clean JSON format for easy parsing
  • Free to use for fan projects
