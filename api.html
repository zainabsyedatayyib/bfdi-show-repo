üöÄ Quick Start

Access the API data directly via:

URL: /searchdata.json Method: GET Content-Type: application/json

The API returns all character data in a single JSON file - no complex endpoints needed!

üìä Response Structure

The API returns a JSON object with the following structure:

{ "code": 200, "status": "OK", "things2search": { "character-key": { "name": "Character Name", "badge": "Role/Status", "desc": "Description", "image": "Image URL" }, ... } }

Response Fields:

  • code: HTTP status code (200 for success)
  • status: Status message (e.g., "OK")
  • things2search: Object containing all character data

üîç Character Data Structure

Each character entry in the things2search object contains:

{ "character-key": { "name": "Character Display Name", "badge": "Contestant/Winner/etc", "desc": "Detailed character description...", "image": "URL or path to character image" } }

Field Descriptions:

  • character-key: Unique identifier (lowercase, hyphenated)
  • name: Character's display name as it appears in the show
  • badge: Character status (e.g., "Contestant", "Winner")
  • desc: Detailed description of the character and their role
  • image: URL or file path to character image

üíª Code Examples

JavaScript (Fetch API)

// Fetch all character data fetch('searchdata.json') .then(response => response.json()) .then(data => { console.log(`Status: ${data.status}`); console.log(`Total characters: ${Object.keys(data.things2search).length}`); // Access specific character const firey = data.things2search.firey; console.log(`${firey.name}: ${firey.desc}`); // List all character names Object.values(data.things2search).forEach(char => { console.log(`${char.name} - ${char.badge}`); }); }) .catch(error => console.error('Error:', error));

JavaScript (async/await)

async function getCharacters() { try { const response = await fetch('searchdata.json'); const data = await response.json(); // Filter contestants only const contestants = Object.entries(data.things2search) .filter(([key, char]) => char.badge === 'Contestant'); console.log(`Found ${contestants.length} contestants`); return data.things2search; } catch (error) { console.error('Failed to fetch characters:', error); } } // Search for a character by name async function searchCharacter(searchTerm) { const characters = await getCharacters(); return Object.values(characters) .filter(char => char.name.toLowerCase() .includes(searchTerm.toLowerCase())); }

üîß Troubleshooting

Q: The API request is failing / 404 error

A: Make sure you're using the correct relative or absolute URL path to searchdata.json. The file is located at the root of this website.

Q: Images are not loading

A: Some character images use external URLs (cdn.comic.studio), while others are local file paths. Check the image field and handle both cases appropriately.

Q: How do I search for a specific character?

A: You can iterate through the things2search object or convert it to an array. Use the character key (lowercase, hyphenated) for direct access, or search by the name field for display names.

Q: Can I modify the data?

A: The JSON file is read-only. You'll need to download it and host your own modified version if you need custom data.
